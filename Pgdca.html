Post Graduate Diploma in Computer Applications (PGDCA)

Advance your IT knowledge and skills for a professional career in technology.

About the Course

The PGDCA is a 1-year postgraduate diploma program designed for graduates who want to enhance their career in the field of computer applications. This course focuses on advanced programming, database systems, software engineering, and IT management.

Course Details

Duration: 1 Year (2 Semesters)

Eligibility: Graduate in any discipline (preferably with Mathematics/Computer Science background)

Why Choose PGDCA?

Advanced Programming & Database Concepts
Specialization in Software Engineering
Career Opportunities in IT & Software Firms
Pathway to MCA, M.Sc. (CS), or Higher Studies

Subjects / Curriculum

  • Fundamentals of Information Technology
  • Programming with C and C++
  • Database Management Systems (DBMS)
  • Software Engineering
  • Web Technologies (HTML, CSS, JavaScript, PHP)
  • Java Programming
  • Data Structures & Algorithms
  • Operating Systems
  • Project Work / Internship
